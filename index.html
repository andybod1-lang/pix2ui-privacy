Pix2UI ("we", "our", "us") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, and safeguard your information when you use our browser extension.

1. Information We Collect

1.1 Screenshot Data

When you capture a webpage element, the screenshot is:

  • Processed in real-time by the AI service (Anthropic Claude)
  • NOT stored on any servers
  • Transmitted securely via HTTPS
  • Deleted immediately after code generation

1.2 API Keys (BYOK Model)

Pix2UI uses a "Bring Your Own Key" model:

  • You provide your own Anthropic API key
  • Your API key is stored locally in your browser only
  • We never see, store, or transmit your API key
  • Screenshots are sent directly from your browser to Anthropic

1.3 Usage Data

We store locally on your device:

  • Number of conversions used this month (free tier)
  • Monthly reset date
  • Your preferences (selected framework, etc.)

This data is stored using your browser's local storage and is never transmitted to our servers.

1.4 Payment Data

If you subscribe to a paid plan:

  • Payments are processed by Stripe via ExtensionPay
  • We do not store credit card information
  • We receive only: email, subscription status, and purchase date
  • Payment data is subject to Stripe's Privacy Policy

2. How We Use Your Information

We use the collected information solely to:

  • Generate code from your screenshots
  • Track your monthly usage quota (free tier)
  • Process subscription payments
  • Provide customer support

We do NOT:

  • Sell your data to third parties
  • Use your data for advertising
  • Store screenshots or generated code
  • Track your browsing history
  • Have access to your API key

3. Third-Party Services

3.1 AI Processing (Anthropic Claude)

Screenshots are sent directly from your browser to Anthropic's Claude API using your own API key.

  • Data is processed under Anthropic's Privacy Policy
  • Anthropic does not use API data to train their models

3.2 Payment Processing (ExtensionPay/Stripe)

  • Payments handled by ExtensionPay
  • Underlying processor: Stripe

4. Data Security

We implement industry-standard security measures:

  • All data transmitted via HTTPS/TLS encryption
  • No server-side storage of screenshots
  • API keys stored only in your browser's secure local storage
  • Direct browser-to-Anthropic communication (no middleman)

5. Your Rights

You have the right to:

  • Access: View your usage data in the extension
  • Delete: Clear local storage via browser settings
  • Portability: Export your data (contact us)
  • Withdraw: Cancel subscription anytime

6. Children's Privacy

Our extension is not intended for children under 13. We do not knowingly collect data from children.

7. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be:

  • Posted on this page with updated date
  • Notified via extension update notes
